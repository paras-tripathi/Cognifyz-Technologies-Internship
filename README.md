# 🍽️ Cognifyz Data Science Internship

## 📌 Overview

This repository contains my work completed during the **Cognifyz Technologies Data Science Internship**. The project focuses on Exploratory Data Analysis (EDA), data preprocessing, visualization, and feature engineering using Python. The tasks are based on a restaurant dataset and provide insights into customer preferences, pricing, restaurant distribution, and online services.

---

## 🚀 Technologies Used

* Python
* Jupyter Notebook (VS Code)
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📂 Internship Tasks

### 🔹 Level 1

#### Task 1 – Data Exploration & Preprocessing

* Loaded and explored the dataset
* Checked dataset dimensions and data types
* Handled missing values
* Analyzed the distribution of Aggregate Rating

#### Task 2 – Descriptive Analysis

* Generated statistical summaries
* Explored categorical variables
* Identified top cities and cuisines

#### Task 3 – Geospatial Analysis

* Visualized restaurant locations
* Studied restaurant distribution across cities
* Analyzed location vs. ratings

---

### 🔹 Level 2

#### Task 1 – Table Booking & Online Delivery

* Calculated service availability percentages
* Compared average ratings
* Analyzed delivery availability across price ranges

#### Task 2 – Price Range Analysis

* Determined the most common price range
* Calculated average ratings for each category
* Visualized rating trends

#### Task 3 – Feature Engineering

* Created new features
* Encoded categorical variables
* Prepared data for further analysis
